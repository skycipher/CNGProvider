Driver Package Signing Outside of the New Build System

In the new WDK build system (MSBUILD) the driver signing task is integrated into the build

toolset. Hence, driver signing can be done seamlessly while building the driver. This document

however, points to various resources that will help understand the concepts of digital signatures for

drivers and how to use the tools involved.

 

Please visit the following link to understand the step by step guidelines in obtaining a “Digital Signature

for Kernel Modules on x64-based Systems Running Windows Vista and above”.
